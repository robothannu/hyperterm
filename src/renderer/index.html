    Groups & Sessions

    A Group is a named container for one or more terminal sessions (panes). Groups persist across app restarts — your work is automatically saved.

    Creating & Managing Groups

    • New Group — Click the + button in the sidebar
    • Rename Group — Double-click the group name in the sidebar
    • Close Group — Click the × button on the group entry
    • Switch Groups — Click a group in the sidebar, or use Cmd+1–Cmd+9

    Split & Arrange Panes

    • Split Horizontally — Right-click → "좌우 분할" or Cmd+D
    • Split Vertically — Right-click → "상하 분할"
    • Close Pane — Right-click → "Pane 닫기"
    • Navigate Panes — Cmd+← Cmd+→ Cmd+↑ Cmd+↓
    • Resize Dividers — Drag the divider between panes

    Pane Header

    • Rename Pane Title — Double-click the pane header title
    • Dead Session Recovery — If a session ends, click "Session ended" overlay to restart it

    Projects / Clusters

    • Set Cluster — Cmd+Shift+G to group related terminals under a project name

    Notes

    • Open Notes — Click the pencil icon ✎ on a group entry
    • Add Note — Type in the notes panel and press Cmd+Enter

    Keyboard Shortcuts

    Cmd+NNew Group
    Cmd+1–Cmd+9Switch to Nth group
    Cmd+← Cmd+→ Cmd+↑ Cmd+↓Navigate panes
    Cmd+Shift+GSet cluster/project name
    Cmd+0Reset font size
    Cmd++Increase font size
    Cmd+-Decrease font size
    Cmd+ASelect all
    Cmd+CCopy (or send SIGINT if no selection)
    Cmd+VPaste
